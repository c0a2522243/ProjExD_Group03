# 正しいこうかとんを探せ!!

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 正しいこうかとんの画像をクリックする
* 参考URL：[サイトタイトル]()

## ゲームの遊び方
* 表示されたこうかとんと同じ画像を見つけてクリックする
* 制限時間を超えたらゲームオーバー

## ゲームの実装
### 共通基本機能
* 背景画像とこうかとんの画像生成

### 分担追加機能
* 制限時間（担当：）：ゲーム全体の制限時間内を設定するクラス
* エンドレス（担当：）：問題をクリアしたら次の問題が出てくる機能
* レベルアップ（担当：）：問題をクリアするにつれ難易度が上がる機能を実装する
* 結果表示（担当：）：どこのステージまでクリアできたのか最終画面で表示する機能
* 
* 効果音追加（担当：）：誤った選択をした際になる効果音を追加する機能
